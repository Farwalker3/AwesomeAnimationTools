Awesome Animation Tools

Pencil2D

A simple, lightweight 2D hand-drawn animation program great for beginners and traditional animators.

OpenToonz

A powerful, studio-grade animation tool used professionally, with many community enhancements.

Anime Effects

A 2D animation tool focused on bones, deformation, and cut-out style animation.

Animation Desk

A user-friendly sketch-to-animation tool popular on tablets and mobile devices.

Animation Paper

Professional drawing-focused animation software designed for clean workflows and precision.

Tahoma 2D

A fork of OpenToonz focused on accessibility and ease of use, great for schools and solo creators.

Synfig

Open-source vector-based 2D animation software with powerful automation tools.

OpenToonz (Alt)

An alternative build of OpenToonz, offering different packaging and versioning.

Friction

An experimental animation tool designed for generative and procedural motion workflows.
